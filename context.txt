PROJECT: ProjectPlanner - Веб-планировщик проектов

TECH STACK:
- Next.js 14 (App Router)
- JavaScript (not TypeScript)
- Tailwind CSS
- shadcn/ui components
- lucide-react icons
- date-fns (для форматирования дат)

ВАЖНЫЕ ПРАВИЛА ДЛЯ LLM:
1. Общайся с пользователем на английском языке
2. Весь UI, текст, кнопки, заголовки должны быть на русском языке
3. Дизайн: темная тема по умолчанию (dark mode)
4. Основной цвет: изумрудный (emerald-500/600) для акцентов
5. Проект фокусируется на минимализме, функциональности и читаемости

СТРУКТУРА ФАЙЛОВ:

Корневые файлы:
- app/layout.jsx - Главный layout с темной темой и AppHeader
- app/page.jsx - Домашняя страница (landing)
- app/globals.css - Глобальные стили

Страницы проектов:
- app/projects/page.jsx - Страница списка проектов
- app/projects/[id]/page.jsx - Страница редактора проекта (заглушка)

Компоненты:
- components/app-header.jsx - Адаптивный хедер с логотипом и профилем пользователя
- components/projects-list.jsx - Главный компонент списка проектов (обновлен)
- components/stat-cards.jsx - Карточки статистики (Всего проектов, Активные и т.д.)
- components/filter-bar.jsx - Панель фильтров и сортировки + кнопка "Новый проект"
- components/project-card-grid.jsx - Карточка проекта (грид-вид)
- components/project-card-list.jsx - Элемент списка проекта (лист-вид)
- components/activity-widget.jsx - Виджет активности (GitHub-style activity graph)
- components/analytics-widget.jsx - Виджет аналитики проектов
- components/activity-squares.jsx - Основной компонент GitHub-style activity graph
- lib/utils.js - Утилиты shadcn

ВАЖНЫЕ ЗАМЕЧАНИЯ ПО КОДУ:
- Используем "use client" для интерактивных компонентов
- Мокаем данные для начала (позже перейдем на localStorage)
- Избегаем memory leaks и оптимизируем re-renders
- Все новые компоненты должны следовать существующему дизайну
- ActivitySquares использует детерминированную генерацию данных для избежания hydration errors
- Конфигурируемая цветовая система: легко менять количество уровней и цвета

КОНФИГУРАЦИЯ ACTIVITY-SQUARES (легко настраивается):
- colorLevelsCount: 8 (количество уровней интенсивности)

- colorIntensityScale: [900, 700, 500, 300, 200, 150, 100, 50] (шкала интенсивности)
